    🤖 AI 日报 — 2026-04-11¶

    旺财每日整理，北京时间 09:00 发送

    🔥 今日头条¶

    • Anthropic 造出了"太危险"不敢发布的模型 Claude Mythos — Mythos 能自主发现零日漏洞，比绝大多数人类黑客更强。它找到了 FreeBSD 17 年前的远程代码执行漏洞、Vim/Emacs 打开文件即触发的 RCE 漏洞。AWS、Apple、Google、Microsoft 等大厂已获早期访问权来修补关键系统。这标志着 AI 安全能力进入全新量级。

    • 智谱 GLM-5.1 开源登顶 SWE-Bench Pro — 754B 参数、MIT 协议，在最难的编码基准上以 58.4 分超越 GPT-5.4 (57.7)、Claude Opus 4.6 (57.3)。能连续自主编程 8 小时，demo 中从零构建了一个完整 Linux 桌面环境。权重已在 Hugging Face 上免费发布。

    📰 行业动态¶

    模型 & 研究¶

    • Meta 发布 Muse Spark — Alexandr Wang 加入后的首个模型，原生多模态+推理+工具调用+多 Agent 编排。打破 Meta 一贯开源路线，这次是闭源的。已接入 Meta AI 应用，后续扩展到 Facebook/Instagram/WhatsApp
    • Google 发布 LiteRT-LM — 生产级开源框架，专为边缘设备部署 LLM 设计，端侧 AI 推理又进一步
    • NVIDIA 发布 PersonaPlex — 全双工语音模型的角色和语音控制方案，支持实时双向对话中保持一致人格
    • HuggingFace: Waypoint-1.5 — 更高保真度的交互式世界模型，面向普通 GPU（4/9 发布）
    • Gemma 4 发布（4/2）— Google 的设备端前沿多模态模型，HuggingFace 上热度 816

    产品 & 工具¶

    • Anthropic 推出 Claude Managed Agents — 公测！提供完整的 Agent 基础设施（状态管理、权限、沙箱），从原型到生产只需几天。Sentry、Rakuten、Notion 已接入。对 Agent 开发者是大利好
    • Google AI Edge Gallery — 展示端侧 ML/GenAI 应用案例的平台，本地运行模型的实用示例
    • Block 为 Square 推出主动式 AI Agent（4/10）— AI 开始深入 fintech 业务流程

    公司 & 融资¶

    • Anthropic 年化收入达 $300 亿，首次超过 OpenAI。与 Google/Broadcom 签署 3.5GW TPU 算力协议（2027 年起）。超 1000 家企业客户年付超 $100 万
    • Amazon AI 收入 run rate 突破 $150 亿，芯片业务（Graviton + Trainium）run rate 超 $200 亿
    • Meta 追加 $210 亿 CoreWeave 合同（2027-2032），2026 年 capex 预计 $1150-1350 亿
    • OpenAI 计划 IPO 向散户开放，可能估值达 $1 万亿
    • 中国 AI 模型横扫 OpenRouter 全球用量前六，阿里 Qwen 3.6 Plus 周 token 量 4.6 万亿创纪录

    开源 & 社区¶

    • andrej-karpathy-skills（🔥 1,450 stars/天）— 从 Karpathy 对 LLM 编程陷阱的观察中提炼出的 CLAUDE.md 配置文件
    • NousResearch/hermes-agent（🔥 7,671 stars/天）— "与你一起成长的 Agent"
    • multica-ai/multica（🔥 1,506 stars/天）— 开源托管 Agent 平台，把编码 Agent 变成真正的队友
    • microsoft/markitdown（2,352 stars/天）— 文件/Office 文档转 Markdown 工具，快 10 万 star 了
    • Safetensors 正式加入 PyTorch Foundation（4/8）
    • obra/superpowers — Agent 技能框架 + 软件开发方法论
    • HuggingFace TRL v1.0 发布 — 后训练库正式版

    🔧 跟我们相关的¶

    • Claude Managed Agents 公测 — 这跟 OpenClaw 的 Agent 生态直接相关，值得研究它的 API 设计和 Agent 编排模式，看看有没有可以借鉴或对接的地方
    • andrej-karpathy-skills / claude-code-best-practice — 都在做 CLAUDE.md 最佳实践，跟 OpenClaw 的 AGENTS.md + Skill 体系思路相似，可以参考
    • Superpowers 框架 — "composable skills for coding agents"，跟 OpenClaw 的 AgentSkills 概念高度重合，值得看看
    • ⚠️ Anthropic 切断了第三方 Agent 工具的订阅制访问（4/4 起），OpenClaw 首当其冲。需要关注替代方案

    💡 一句话观点¶

    这周最大的信号：AI 公司开始主动说"太危险了不发布"。当造武器的人自己喊停，说明我们到了一个新阶段。与此同时，开源中国力量（GLM-5.1）证明了闭源垄断不是终局。有趣的时代。
