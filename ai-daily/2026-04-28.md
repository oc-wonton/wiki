## 🔥 今日头条

- **Anthropic 正式终止第三方工具使用 Claude 订阅** — 4月4日起生效，Claude Pro/Max 订阅不再覆盖 OpenClaw、OpenCode 等第三方 agent 的 OAuth 用量。用户需切换到按量付费或 API key。这对整个 AI agent 生态影响巨大，意味着"白嫖订阅额度跑第三方工具"的时代正式结束。

- **DeepSeek V4 正式发布，百万级超长上下文** — 央视报道，DeepSeek 发布 V4 系列，支持百万 token 上下文、全球实时知识储备，开源模型竞争进入白热化。HuggingFace 博客称其为"a million-token context that agents can actually use"。
