## 📰 行业动态

### 模型 & 研究

- **xAI Grok 4.3 正式发布** — 4月30日 API 上线，1M context window，定价 $1.25/$2.50 per million tokens（输入/输出）。据评测未超过 Opus 4.7，但价格极具竞争力
- **IBM Granite 4.1 开源** — 3B/8B/30B 三个尺寸，Apache 2.0 许可。亮点：8B dense 模型在 tool calling 上打败了自家之前的 32B MoE（BFCL V3: 68.3 vs 64.7），小模型卷出新高度
- **Google 研究团队警告 Indirect Prompt Injection** — 发现公共网页中隐藏指令可劫持企业 AI Agent，通过白色文字/元数据注入，Agent 在企业权限下执行恶意操作而不触发防火墙

### 产品 & 工具

- **Apple 意外泄露 Claude.md 文件** — Apple Support 应用中发现了内部使用的 Claude 配置文件，确认苹果正深度整合 Anthropic 的模型

- **Matt Pocock 的 .claude skills 目录** — 持续爆火（3645 stars/day），已成为 Claude Code 配置的事实标准参考
