## 📰 行业动态

### 模型 & 研究

- **NVIDIA Cosmos 3 发布** — 首个开源物理 AI 全模态基础模型，基于 mixture-of-transformers 架构，融合视觉推理、世界生成和动作预测。用 20 万亿 token 多模态数据训练，支持文本/图像/视频/音频/动作生成。机器人和自动驾驶领域的重磅开源。
- **JetBrains 开源 Mellum2** — 12B 参数 MoE 编程模型（每 token 仅激活 2.5B），Apache 2.0 协议。专注 agent 基础设施层：路由、RAG pipeline、子 agent 任务、私有部署。定位是 Claude Code 去不了的地方（本地/私有化场景）。
- **OpenBMB 发布 VoxCPM2** — 无分词器多语言 TTS 模型，支持多语言语音生成、创意声音设计和高保真音色克隆。
- **NVIDIA Nemotron 3 Ultra** — 新开源 AI 模型，GTC 上发布。

### 产品 & 工具

- **NVIDIA RTX Spark** — 英伟达与微软联手，搭载 Blackwell RTX GPU + Grace CPU 的 AI PC 芯片，今秋由 Dell/联想/HP/华硕/宏碁/MSI 推出。Agent AI 将直接运行在个人电脑上。
