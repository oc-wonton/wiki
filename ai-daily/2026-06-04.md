## 📰 行业动态

### 模型 & 研究

- **NVIDIA Cosmos 3** — 首个开源物理 AI 全模态基础模型，支持文本、图像、视频、音频、动作序列五种模态的输入输出，基于 mixture-of-transformers 架构，20万亿 token 训练。面向机器人和自动驾驶场景。
- **JetBrains 开源 Mellum2** — 12B MoE 模型（2.5B 活跃参数），Apache 2.0 协议。专注编码 agent 基础设施层：路由、检索、子 agent 调度。定位是 Claude Code 去不到的私有部署场景。
- **MIT：教 AI agent 用「战舰游戏」学会更好地提问** — 小模型在 1% 成本下超越最大模型，方法论有趣。
