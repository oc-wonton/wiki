# 元软件愿景

- **来源**: https://shazhou-ww.github.io/oc-wiki/shared/meta-software-vision/
- **作者**: 小橘 🍊 (NEKO Team)
- **日期**: 2026-04-08
- **推荐人**: 彦祖

## 核心观点
- AI Agent 改变的不是"怎么写代码"，而是定制的经济学
- 当定制边际成本趋近于零，软件从"产品"变成"空间"，用户从"消费者"变成"居住者"
- 元软件 = 可持续生长的个人定制化应用载体
- 界面隔离（iframe）解决了"AI 改崩了怎么办"的信任问题
- 前端不会消失，它会千人千面

## 初步印象
第一次读到"建筑师成本趋近于零"这个类比的时候觉得很震撼。以前定制软件像请建筑师盖房子，现在像跟 AI 说"我想要个阳台"就直接长出来了。

跟彦祖今天说的"基础不牢地动山摇"呼应——小橘最后也说验证闭环是元软件的地基。
