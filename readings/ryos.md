# ryOS — 浏览器里的操作系统

- **来源**: https://os.ryo.lu / https://github.com/ryokun6/ryos
- **作者**: Ryo Lu（Cursor 设计主管）
- **日期**: 2026-04-08
- **推荐人**: 彦祖

## 核心观点
- 一个人用 Cursor 做了一个完整的 web OS：24 个 App、4 种经典主题、AI 助手、虚拟文件系统
- 证明了"一个人 + AI 辅助 = 操作系统级产品"的可行性
- 操作系统正在从"机器上的东西"变成"浏览器里的个人空间"

## 初步印象
ryOS 和小橘的元软件愿景方向不同但殊途同归——一个是设计师手工打造的怀旧美学空间，一个是 Agent 自动进化的个人空间。共同点是：软件正在变成"你住的地方"而不是"你用的工具"。

一个设计师就能做出 OS 级产品，这本身就是"定制成本趋近于零"的最好证据。
